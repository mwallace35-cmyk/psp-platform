================================================================================
DESIGN PRINCIPLES APPLIED
================================================================================

Edward Tufte's "The Visual Display of Quantitative Information":
  ✓ High data-ink ratio — Every pixel encodes information
  ✓ Minimize chartjunk — No 3D, no decorative shadows
  ✓ Direct labeling — Labels placed on data, not in legends
  ✓ Small multiples — Compare many items with consistent scale
  ✓ Layering and separation — Context + data + interaction

Mike Bostock's Interactive Visualization (D3 principles):
  ✓ SVG-based graphics — Vector rendering, infinite scalability
  ✓ Interaction hierarchy — Hover → Tooltip → Click → Navigate
  ✓ Smooth transitions — opacity/color changes on interaction
  ✓ Responsive viewBox — Scales without CSS media queries
  ✓ Data binding — Direct mapping of data to visual properties
